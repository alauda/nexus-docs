## v3.81.1

### Features and Enhancements

#### Alauda Build of Nexus

Includes Alauda Build of Nexus v3.81.1 operator that deploys Nexus Repository Manager Community Edition.

With this release, bringing enhanced stability and bug fixes. To know more about Nexus Repository Manager v3.81 features, please visit [Nexus Repository Manager v3.81.1 Release Notes](https://help.sonatype.com/en/sonatype-nexus-repository-3-81-0-release-notes.html).

:::info
**Important:** Nexus Community Edition limitations

It supports up to **100,000 total components** and **200,000 requests per day**. Exceeding these limits will prevent adding new components. For more details, see [Community Edition Limitations](https://help.sonatype.com/en/ce-onboarding.html#community-edition-limitations).
:::
