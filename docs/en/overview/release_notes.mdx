::: danger Deprecated: Nexus 3.81 (nexus-ce-operator v3.81.1 and v3.81.2)

Due to restrictive limitations in Nexus 3.81 Community Edition, **this version has been discontinued**. Once usage reaches the quota limits (100,000 total components and 200,000 requests per day), the instance will cease to function properly.

**If you have already upgraded to version 3.81, please follow the [Rollback to Nexus 3.76](../howto/04_rollback_to_3.76.mdx) guide to downgrade.**

For more information about Community Edition limitations, see [Community Edition Limitations](https://help.sonatype.com/en/ce-onboarding.html#community-edition-limitations).

:::

# Release Notes

::: tip
The maintenance period for version **v3.81.z** is from August 15, 2025 to August 15, 2026. 
:::

## Compatibility and support matrix

The following table shows the compatibility and support matrix between the Alauda Build of Nexus operator and ACP versions.

| Alauda Build of Nexus Operator Version | Nexus Version |  ACP Version |
| --------------------------------- | --------------- | --------------- |
| v3.81.2                           | v3.81.1         | V4.0,V4.1 |
| v3.81.1                           | v3.81.1         | V4.0,V4.1 |
| v3.76.7                           | v3.76.0         | V4.0 |

:::info
**Important:** Nexus Community Edition limitations

It supports up to **100,000 total components** and **200,000 requests per day**. Exceeding these limits will prevent adding new components. For more details, see [Community Edition Limitations](https://help.sonatype.com/en/ce-onboarding.html#community-edition-limitations).
:::
