# Release Notes

## v3.76.0

### Features and Enhancements

#### Alauda Build of Nexus

Introduces Alauda Build of Nexus v3.76.0 Addon that deploys Nexus Repository Manager Community Edition. 

Includes deployment using the following templates:

- Quick Start: A template for quickly creating a Nexus instance, suitable for development and testing scenarios, not recommended for production environments.
- Production: A template for creating a production-ready Nexus instance, recommended for production environments.


To know more about Nexus Repository Manager v3.76 features, please visit [Nexus Repository Manager v3.76.0 Release Notes](https://help.sonatype.com/en/sonatype-nexus-repository-3-76-0-release-notes.html#what-s-new-in-3-76-0-).
